Hi, I'm Andrés.

A colombian graphic designer and I created this landing page to learn html 5, css 3 and bootstrap 5 using, as an excuse, a behind the scenes of this cyborg in virtual clothing I created with geometry nodes in Blender 3d 3.1, following the tutorial Generative Modeling Crash Course for Blender! by Curtis Holt, and textured in Adobe Photoshop 2022.

Some inspiration.

The mood board was filled with works under the tags #virtualfashion and #metafashion on Instagram plus early 90's brand color schemes.
